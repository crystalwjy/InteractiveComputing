Snake Game

Overview

This project was inspired by the classic snake game, where the user can control the snake to eat food and earn points. I want to recreate it because I sometimes saw people playing it in my other classes. I was also a player of Snake.io when it was popular back in high school time. Overall I think this game is easy to play and addictive.

The design of this game is relatively simple. The user can only control the snake by pressing W/A/S/D to eat the food. The score will add one point when the user collects the food. If the snake hits the border, then the game ends. This game has three difficulty levels, which refer to different snake movement speeds. It will be more challenging to control the snake when it moves faster.
