Stardew Valley Recreation Proposal

Group Members: Isabel Chai & Crystal Wu

- PROPOSAL -

Since both of us had experiences of playing Stardew Valley before and were really interested in this game, we planned to recreate this game with the VR features. We may not create the whole six maps in the original game. We planned to choose only one map to develop, which probably is the standard farm or the beach farm. This recreation will also allow players to plant vegetables, fruits, and trees as well as fishing, raising animals and interacting with NPCs. The game mechanisms will be similar to the original game, in which the characters will die if they were beaten by the monsters and they will lose energy through working. The aesthetic style will still be pixel style, but the appearances of characters and the farm may change a little.

We will use an array to store the tools and the objects, such as the seeds and fishes, that the characters obtain in the game and display them at the bottom of the screen. The inventory will also indicate whether the player is holding that object or not and will remove that object once the player has moved them to other storage places–such as the boxes–or the objects have been used up. The number of NPCs in our project may not be as many as those in the original games. We will select several essential ones such as Louis, Pierre, and Marnie to ensure that the players can buy and sell things as well as get essential items to develop the story lines. Moreover, this project will also allow the players to decorate their cabins and farm houses, but they cannot change the outside design of the buildings.

Different from the version on PCs, the VR version of this game will change the view from the third-person point of view to the first person. The player will see their hands empty or holding objects (tools or objects) and interact with other players or NPCs face to face. The microphone will record some simple sentences from the users such as saying “Yes” or “No” to communicate with others.

Project Update #1

This week, we created the start and character creation pages. For the start page, we used the original image from Stardew Valley. The user has the following functionality for the character creation page: input name, farm name, and favorite thing. On the left is an image where the user can control different values for visual elements. Users can adjust skin, hair, shirt, pants, and accessories. They can also adjust the color of these visual features with the slider.

For the next step, we plan to do the following:
1. Write the storyline to introduce the game
2. Create the home map for the user to move, click, use tools, open new window
3. Create a list of activities: farming, fishing, buying items, building relationships with other people, raising animals
4. Create a store page for users to spend coins and purchase items

Project Update #2

This week, we successfully stored the user input of name, farm name, and favorite thing and displayed the user input in the introduction dialogues. We also created the overhead map and the player inside the canvas, so the user can always be at the center of the screen, and the map will move with a specific offset. In addition, we have a basic user home, which the user can move around and switch tools at the bottom of the screen. For the next step, we plan to do the color-based collision to check if the user can enter a specific part of the map.
