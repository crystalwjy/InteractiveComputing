Assignment 07: Fries Only!

Welcome to the game Fries Only! This game was created due to my love for McDonald's fries. You will get 1 point by eating the fries, but also deduct 1 point if you eat the hamburger. The position of both fries and hamburger will change randomly with differentt frames.
This game uses live video and uses machine learning for feature tracking. It uses nose tracking and the mouth on the screen will move with the position detected with the user nose.
